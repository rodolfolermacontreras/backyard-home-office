# Foundation

Foundation design, leveling procedure, and anchoring details.

## Overview

No slab, no excavation. 6 adjustable feet sit on any stable surface.

## Surfaces

- Concrete: set feet directly, no prep
- Gravel: compact and level gravel first
- Grass: compact soil, use foot pads
- Slope: up to 5° handled by adjustable feet

## Leveling Procedure

1. Mark the 6 foot positions per the layout drawing
2. Place feet, extend to approximately level
3. Set floor panel and check with level
4. Adjust feet until all axes are level (±1/8" tolerance)
5. Torque lock nuts to spec

## Anchoring

Optional ground anchors for high-wind zones. See `anchor-kit.md` for hardware.
