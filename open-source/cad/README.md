# CAD

3D design files for the WorkPod Pro structure and components.

## Folders

```
concept/    Early sketches and ideation (not buildable, reference only)
source/     Editable CAD files — keep in original format
step/       STEP exports — works in any CAD tool
stl/        STL exports — for 3D printing small parts
drawings/   2D drawings with dimensions (PDF)
```

## Parts

- `shell-exterior/` — outer wall and roof panels
- `shell-interior/` — inner wall panels and ceiling
- `foundation-frame/` — pedestal base and foot mounting
- `door-assembly/` — door frame, hinges, hardware
- `window-assembly/` — frame, glass mount, seals
- `furniture/` — built-in bookshelf, desk mount options
