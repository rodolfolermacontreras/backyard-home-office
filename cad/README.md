# CAD

3D design files for all pod variants.

## Folders

```
pod-pro/          WorkPod Pro
  step/           STEP exports (opens in any CAD tool)
  stl/            STL exports for 3D printing
pod-versatile/    Pod Versatile (Studio Pod)
  step/
  stl/
pod-core/         Pod Core
  step/
  stl/
```

## File Naming

`part-name_v1.stp`, `panel-exterior-left_v2.stl`

## Contributing CAD

- Always export and upload `.stp` to the relevant `step/` folder
- Export `.stl` to the relevant `stl/` folder for 3D printing
