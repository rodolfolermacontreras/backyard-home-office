# CAD

3D design files for the WorkPod Pro and Studio Pod.

## Folders

```
step/           WorkPod Pro — STEP exports (works in any CAD tool)
stl/            WorkPod Pro — STL exports for 3D printing
studio-pod/     Studio Pod variant
  step/
  stl/
```

## File Naming

`part-name_v1.stp`, `panel-exterior-left_v2.stl`

## Contributing CAD

- Always export and upload `.stp` to the relevant `step/` folder
- Export `.stl` to the relevant `stl/` folder for 3D printing
