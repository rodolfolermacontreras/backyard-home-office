# WorkPod Pro — CAD Files

Mid-sized backyard office. More room to stretch out, sloped ceiling, premium design.

| | |
|---|---|
| Floor area | 102 sqft |
| Ceiling height | 6.8'–9.3' sloped |
| Foundation | 6 adjustable feet, 5° slope |
| Weight capacity | 2.9 tons |
| Comfort range | 45–100°F |
| Electrical | 2 wall outlets, 1 floor outlet, 1 ceiling light, 1 ventilator |

| Installation | 1–2 days, no slab required |

## Folders

- `step/` — STEP files (opens in Fusion 360, FreeCAD, SolidWorks, etc.)
- `stl/` — STL files for 3D printing or previewing
